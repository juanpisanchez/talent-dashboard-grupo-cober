ℹ️ Cómo publicar en GitHub Pages
1. Creá un repositorio en GitHub (puede ser privado o público).
2. Subí este archivo HTML como index.html en la rama main.
3. En el repositorio → Settings → Pages → Source: Deploy from branch → main → /root.
4. GitHub te dará una URL tipo https://tu-usuario.github.io/nombre-repo.
5. Compartí esa URL con tu equipo. El dashboard se actualiza solo al recargar la página.
